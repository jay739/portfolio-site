# Contributing to portfolio-site

Thanks for considering a contribution. Issues, discussions and PRs are all
welcome.

## Getting set up

```
git clone https://github.com/jay739/portfolio-site && cd portfolio-site
npm install
npm run dev
```

Next.js + TypeScript. Node 20+.

## Running the checks

```
npm run lint
npm test
```

## Ground rules

- Open an issue or discussion before large changes, so effort isn't wasted.
- Keep PRs focused: one change per PR.
- New behavior needs a test where the repo has a test suite.
- CI must pass before merge.
